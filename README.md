## features ‚ú®

- Vue
  - reactivity
  - composition api
  - lifecycle hooks
  - class and style normalize
  - custom renderer
- React
  - JSX
  - stateful functional component
  - props.children (i.e., slots in Vue)
  - render props
- no dependencis, all tools homemade
  - JSX parser (extend the React JSX)
  - assets bundler (no bundling, use native ES modules in the browser) üì¶
